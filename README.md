# Eat-N-Split

A React application that helps users split bills with friends.

## Features

- Add new friends
- Select a friend
- Split bills
- Controlled form components
- Derived state
- Component composition
- State lifting

## Built With

- React
- JavaScript
- Vite
- CSS

## What I Learned

- useState
- Controlled Components
- Derived State
- Lifting State Up
- Props
- Component Composition

## Installation

```bash
npm install
npm run dev
```
